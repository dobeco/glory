# Vue.js仿王者荣耀移动端 + 后台管理系统 

## 开启项目

```
git clone https://github.com/dobeco/glory.git

cd glory 
# 后台
cd admin
yarn install
yarn serve

# 手机端
cd yarn install
yarn serve

# 服务端
cd server
npm install
npm start

# 打包运行
分别在admin web目录下运行yarn run build 打包路径已修改，

到server目录下运行npm start浏览器打开http://localhost:9000 

后台地址http://localhost:9000/admin

```
